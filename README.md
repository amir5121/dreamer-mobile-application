# Dreamer

An opensource Flutter app for tracking dreams.

## Getting Started

### Install

- Install Flutter and android studio
- Generate build files with

  `flutter pub run build_runner build --delete-conflicting-outputs`
- To build apk files

  `flutter build apk --split-per-abi`
